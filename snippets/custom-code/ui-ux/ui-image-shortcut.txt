Shortcut button definitions for ui-image-shortcut.js
=====================================================
Paste these into the "Shortcut Buttons" field of your Perchance character.
Each property is on its own line. Separate buttons with a blank line.

The /ai, /user, and /nar commands are handled natively by Perchance.
The /image command is handled by ui-image-shortcut.js.

Usage for /image:
  - Click 🖼️ Image to pre-fill /image --num=3 in the input box.
  - Edit --num=N to any value between 1 and 6 before sending.
  - Send to generate N images from the current conversation context.
  - Omit --num entirely to generate 1 image.

Usage for /ai, /user, /nar:
  - Click the button to pre-fill the command in the input box.
  - Type an optional writing instruction after the command, then send.

-------------------------------------------------------------------------------

@name=🗣️ {{char}}
@message=/ai <optional writing instruction>
@insertionType=replace
@autoSend=no
@clearAfterSend=no

@name=🗣️ {{user}}
@message=/user <optional writing instruction>
@insertionType=replace
@autoSend=no
@clearAfterSend=no

@name=🗣️ Narrator
@message=/nar <optional writing instruction>
@insertionType=replace
@autoSend=no
@clearAfterSend=no

@name=🖼️ Image
@message=/image --num=3
@insertionType=replace
@autoSend=no
@clearAfterSend=no
