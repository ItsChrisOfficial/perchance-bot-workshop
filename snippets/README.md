# snippets

## Purpose

Store reusable snippets for quick composition and experimentation.

## What belongs here

- Frontend snippets
- Backend snippets
- Python snippets
- JavaScript snippets
- General utility snippets

## What does not belong here

- Full bot projects
- Shared production utilities better placed in `shared/utilities/`

## Naming rules

- Use descriptive kebab-case names.
- Keep snippets small and focused.

## Agent workflow rules

- Add snippets to the correct category folder.
- Update `SNIPPETS_INDEX.md` when adding or removing snippets.
