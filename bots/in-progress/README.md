# bots/in-progress

## Purpose

Active workspace for bot development.

## What belongs here

- Draft bot folders
- Iteration notes and temporary bot-specific docs

## What does not belong here

- Final approved bots (move those to `../completed`)
- Global shared utilities or snippets

## Naming rules

- One folder per bot in kebab-case.
- Use stable names; avoid `temp`/`final` naming patterns.

## Agent workflow rules

- Start new bot work here.
- Keep each bot self-contained.
- Move validated bots to `../completed` and update `BOT_CATALOG.md`.
