# agents

## Purpose

Guidance for AI and automation agents working in this repository.

## What belongs here

- Agent workflow notes
- Agent operation checklists
- Agent coordination docs

## What does not belong here

- Bot source/export files
- Reusable snippets
- Shared runtime utilities

## Naming rules

- Use kebab-case file and folder names.
- Keep documents short and task-oriented.

## Agent workflow rules

- Follow `AGENTS.md` and `.github` instruction files.
- Keep diffs minimal.
- Update indexes when adding major assets.
